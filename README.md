P1 Member Access System
========================

Getting Started with Vagrant
============================

    vagrant up

Creating a user
===============

    vagrant ssh
    cd /vagrant
    ./manage.py create_person

Viewing the site
================

open a browser to http://127.0.0.1:8001 on the host machine
